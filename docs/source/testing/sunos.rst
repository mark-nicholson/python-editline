Quirks
------

*  libffi is in place by default, but headers are not in /usr/include (I
   symlinked them -- they are on the system, just not in the correct
   place)
*  you need to manually install GAWK >= version 4.0 to be able to build `thrysoee.dk`_

.. [1] I have added some upgrades (mainly autoconf stuff) to a clone of `thrysoee.dk`_

.. [2] A BusError was seen during testing, but was not reproducible

.. [3] A SegFault was seen during testing, but was not reproducible
